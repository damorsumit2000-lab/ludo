# 🎲 LUDO — Modern Minimal

A clean, modern Ludo game built as a single HTML file. Playable in any browser, deployable on GitHub Pages in one click.

## Features

- ✅ 2 or 4 player support
- ✅ Play vs AI or local multiplayer (or both)
- ✅ Smart AI with capture & strategy logic
- ✅ Full Ludo rules (roll 6 to enter, capture opponents, extra turn on 6)
- ✅ Modern minimal dark UI
- ✅ Mobile responsive
- ✅ Zero dependencies — pure HTML/CSS/JS

## Play Online

👉 **[Play on GitHub Pages](https://YOUR_USERNAME.github.io/ludo)**

## Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Save — your game will be live at `https://YOUR_USERNAME.github.io/ludo`

## Local Dev

Just open `index.html` in your browser. No build step needed.
